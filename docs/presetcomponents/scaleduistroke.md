## ScaledUIStroke
* Type: Preset Component
* Since: 0.0.1

## Construction
```
From "ScaledUIStroke"
```

## Usage
`ScaledUIStroke` is a preset component that makes a given `UIStroke` scale based on the user's device resolution.

You can use it like so:
```lua
New(UIStroke, {
    -- Properties
}, From "ScaledUIStroke")
```
